# Roadmap

## Now

- Запустить первый локальный движок.
- Использовать вход администратора на `/admin`.
- Редактировать страницы через защищённый маршрут `/edit`.
- Хранить рабочие данные в SQLite.
- Подключить Telegram bot token.
- Импортировать посты в черновики.
- Включить автопубликацию Telegram, если сайту подходит публикация без ручной проверки.
- Подключить OpenAI-compatible API для AI-помощника.
- Заполнить базу знаний.

- Медиа-библиотека.

# Project Vision

Мы создаем собственный CMS-движок для быстрых клиентских сайтов.

Главная идея: админка должна быть проще WordPress, но сильнее в интеграциях, автоматизации публикаций и AI-помощниках.

Ключевые направления:

- публикация постов из Telegram;
- подключаемые CRM-интеграции, начиная с Bitrix24;
- база знаний для каждого сайта;
- AI-помощник, который отвечает клиентам сайта;
- мультисайтовость;
- расширяемый слой интеграций.
